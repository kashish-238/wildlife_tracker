Project Description

The Wildlife Sighting Tracker is a web application built with Express.js that loads wildlife observation data asynchronously from a JSON file and provides several API endpoints to filter, search, and analyze sightings.

Available API Endpoints

  • /api/sightings
  • /api/sightings/verified
  • /api/sightings/species-list
  • /api/sightings/habitat/forest
  • /api/sightings/search/eagle
  • /api/sightings/find-index/moose
  • /api/sightings/recent
